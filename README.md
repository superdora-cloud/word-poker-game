# 🎴 ことばポーカー

ひらがなカードで役を作って得点を競う、新感覚のカードゲームです！

## 🎯 ゲームの目的

3ラウンドを通して、手札のひらがなカードで役を作り、最も高い得点を獲得することを目指します。

## 👥 プレイ人数

2人以上で遊べます。デフォルトでは「あなた」と「ともだち」の2人が設定されています。

## 📖 ゲームの流れ

### 1. ゲーム開始前の準備

1. プレイヤー名を入力して「追加」ボタンでプレイヤーを追加
2. 全員が揃ったら「ゲーム開始」ボタンを押す
3. 各プレイヤーに5枚のカードが配られます

### 2. ターンの進行

各プレイヤーのターンは **2つのフェーズ** で構成されています。

#### 🔄 フェーズ1: 手札交換（任意）

- 手札の中から交換したいカードを選択
- 「交換する」ボタンを押すと、選んだカードが新しいカードに交換される
- 交換したくない場合は「交換しない」ボタンで次のフェーズへ

#### 🃏 フェーズ2: カードプレイ

1. 手札から出したいカードを選択
2. 「出す（役判定）」ボタンで役を判定し、得点を獲得
3. **重要**: 1ターンに複数回カードを出せます！
   - 例: 2枚でワンペアを出した後、残りのカードで別の役を作ることも可能
   - ツーペア（2枚×2回）やフルハウス（3枚+2枚）など、戦略的に分けて出すことで高得点を狙えます
4. すべてのカードを出し終わったら「ターン終了」ボタンを押す

### 3. ゲームの終了

- 全プレイヤーが **3ラウンド** を終えると、ゲーム終了
- 最も得点が高いプレイヤーの勝利！

## 🏆 役と得点

| 役 | 条件 | 得点 |
|---|---|---|
| **ファイブカード** | 5枚出す | +15点 |
| **フォーカード** | 4枚出す | +10点 |
| **スリーカード** | 3枚出す | +5点 |
| **ワンペア** | 2枚出す | +2点 |
| **1枚のみ** | 1枚出す | 0点 |

### 💡 得点のコツ

- **1枚だけ出しても得点にならない**ので注意！
- **複数回に分けて出すことで合計点を稼げる**のが「ことばポーカー」の特徴です
  - 例1: ツーペア → 2枚を2回出して合計 **+4点**
  - 例2: フルハウス → 3枚（+5点）+ 2枚（+2点）で合計 **+7点**
  - 例3: 5枚を一度に出せば **+15点**、でも3枚（+5点）+ 2枚（+2点）なら **+7点**... 戦略を考えよう！

## 🎮 操作方法

### ゲーム開始時
- **プレイヤー追加**: 名前を入力して「追加」ボタン
- **ゲーム開始**: 「ゲーム開始」ボタン
- **リセット**: 「リセット」ボタンで最初からやり直し

### ゲーム中
- **カード選択**: カードをクリックして選択（もう一度クリックで解除）
- **交換する**: 選んだカードを新しいカードに交換
- **交換しない**: 交換フェーズをスキップ
- **出す（役判定）**: 選んだカードで役を判定し得点を獲得
- **選択クリア**: 選択したカードをすべて解除
- **ターン終了**: 現在のターンを終了し、次のプレイヤーへ

### ゲーム終了後
- **もう一度プレイ**: ゲームをリセットして最初から

## 📝 ゲームのルール詳細

### デッキについて
- ひらがな50音（あ〜ん、ー）のカードを使用
- **プレイヤー人数に応じてデッキ枚数が自動調整**されます
  - 例: 2人なら各文字2枚ずつ、3人なら各文字3枚ずつ

### カードの補充
- ターン終了時に、使用したカードが手札から除かれます
- デッキに残りがあれば、自動的に5枚になるまで補充されます

### 使用済みカード
- 一度出したカードは **グレー表示** になり、そのターン中は再度使えません
- ターン終了時に手札から削除され、新しいカードが補充されます

## 🎨 画面の見方

### ゲーム情報エリア
- **ラウンド**: 現在のラウンド数／全3ラウンド
- **残りカード**: デッキに残っているカード枚数
- **現在**: 現在プレイ中のプレイヤー名

### プレイヤーカード
- **ゴールドの枠**: 現在のターンのプレイヤー
- **左の色付きバー**: 
  - ゴールド: 1位
  - シルバー: 2位
  - ブロンズ: 3位

### ターン情報エリア
- **このターンの得点**: 現在のターンで獲得した合計得点
- **出した役**: これまでにこのターンで出した役の履歴

## 🌟 戦略のヒント

1. **交換を賢く使う**: 同じカードが多い時は交換せず、バラバラな時は交換を検討
2. **複数回出すか一気に出すか**: 5枚一度に出せば15点だが、3枚+2枚なら7点、2枚×2+1枚なら4点... どれが最適？
3. **デッキの残り枚数を確認**: 終盤はカードが少なくなるので、早めに役を作ろう
4. **他のプレイヤーの得点を確認**: 差を縮めるか、逃げ切るか... 状況に応じた戦略を！
